# TaskMuster — App Store Handoff

## What was done this session

Capacitor was added to wrap the existing React + Vite web app in a native iOS container for App Store submission.

**Installed packages:**
- `@capacitor/core`, `@capacitor/cli`, `@capacitor/ios` — core Capacitor stack
- `@capacitor/assets` — icon/splash screen generator
- `typescript` — required for the `.ts` config file

**Files created/generated:**
- `capacitor.config.ts` — Capacitor config (bundle ID, app name, webDir)
- `ios/` — full Xcode project generated by `npx cap add ios`

**Build verified:** `npm run build && npx cap sync ios` runs clean.

---

## Current state

The Xcode project is generated and synced. The app will load in a WebView. What's left before you can submit:

### Blocked on you

| Task | Details |
|------|---------|
| **App icon** | Create `resources/icon.png` — 1024×1024 PNG, no transparency, no rounded corners. Then run `npx capacitor-assets generate --ios`. |
| **Splash screen** | Create `resources/splash.png` — 2732×2732 PNG. Same command generates it. |
| **Apple Developer account** | Enroll at developer.apple.com ($99/yr). Takes up to 48 hours to activate. |
| **Xcode** | Install from Mac App Store if not already installed (~10GB). |

### Once you have the above

```bash
# Open in Xcode
npx cap open ios

# In Xcode:
# App target → Signing & Capabilities → set Team to your Apple Developer account
# Bundle ID is already set: com.shanerobihemed.taskmuster
```

### Then for submission

1. Create the app record in App Store Connect (appstoreconnect.apple.com)
2. Register bundle ID `com.shanerobihemed.taskmuster`
3. Fill in app name, description, category (Productivity or Health & Fitness), screenshots, keywords
4. Add a privacy policy URL (required — a simple hosted page is fine)
5. In Xcode: select **Any iOS Device** → **Product → Archive** → **Distribute App → App Store Connect → Upload**
6. Submit for review from App Store Connect (review takes 1–3 days)
